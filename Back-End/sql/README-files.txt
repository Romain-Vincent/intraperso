Fichiers SQL:

- Intra.mwb: fichier MySQL Workbench representant le modele de la base de donnees.

- Database.sql: fichier SQL contenant toutes les definitions des tables de la base de donnees.
- Data.sql: fichier SQL contenant les datas pour tester l'application

- add_status_point.sql: stored procedure qui permet d'ajouter une soutenance a un projet.
- add_user.sql: stored procedure qui permet d'ajouter un etudiant a un projet
- drop_user.sql: stored proceudre qui permet de supprimer un etudian d'un projet
- get_status_points.sql: stored procedure qui permet de recuperer toutes les soutenances d'un projet
- is_valid_project.sql: stored procedure pour l'insertion d'un projet - non valide
- is_valid_user.sql: stored procedure qui teste l'existence d'un utilisateur lors du login
